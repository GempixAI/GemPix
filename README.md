## Complete Prompt Engineering Tutorial

While scene-based design simplifies operations, mastering prompt techniques can still significantly improve generation quality. On [GemPix platform](https://gempix-ai.com/), you can test these techniques in real-time.

### Basic Syntax Rules

**Structured Prompt Template**:
```
[Subject description] + [Style modifiers] + [Environment background] + [Lighting atmosphere] + [Quality tags]
```

**Example Analysis**:
```
"a majestic dragon (subject)
in fantasy art style (style)
flying over mountain peaks (environment)
with golden sunset lighting (lighting)
highly detailed, 4k, masterpiece (quality)"
```

### Keyword Weight Techniques

In Gemini model, word position and repetition affect weight:

**Weight Control Methods**:
1. **Position Weight**: Earlier words have higher weight
2. **Repetition Emphasis**: Repeating keywords increases weight
3. **Description Detail**: Detailed description = higher weight

**Practical Comparison**:
```
❌ Inefficient: "a cat, cute, fluffy"
✅ Efficient: "a fluffy white Persian cat,
         extremely cute and adorable,
         soft fur texture"
```

### Using Negative Prompts

Negative prompts tell AI what elements to avoid:

**Common Negative Words**:
- Quality issues: `low quality, blurry, pixelated, distorted`
- Anatomical errors: `deformed hands, extra fingers, missing limbs`
- Style issues: `cartoon (if realistic wanted), realistic (if cartoon wanted)`
- Content issues: `watermark, text, signature`

**Scene-Specific Negative Words**:
- Portrait: `ugly face, asymmetric eyes, bad anatomy`
- Product: `cluttered background, poor lighting, shadows`
- Landscape: `modern buildings (for ancient landscapes), people (for pure landscapes)`

### Pitfall Guide

**Common Mistakes**:

1. **Over-stacking Keywords**
   - ❌ "beautiful amazing gorgeous stunning fantastic awesome incredible"
   - ✅ "beautiful and elegant"

2. **Contradictory Descriptions**
   - ❌ "realistic photo, cartoon style, oil painting"
   - ✅ Choose one main style

3. **Too Vague**
   - ❌ "a nice picture"
   - ✅ "a professional portrait with natural lighting"

4. **Ignoring Negative Prompts**
   - ❌ Only write what you want, not what you don't want
   - ✅ Use both positive and negative prompts

5. **Not Considering Scene Adaptation**
   - ❌ Using fantasy style descriptions in product display scenes
   - ✅ Adjust prompt style based on selected scene

## AI Generated Image Post-Processing Optimization

Generated images often reach high quality, but appropriate post-processing can add the finishing touch.

### Common Post-Processing Tools

**Basic Adjustments**:
- **Lightroom/Photoshop**: Color correction, exposure adjustment
- **GIMP** (free): Open-source alternative
- **Canva**: Quick online editing

**Advanced Optimization**:
- **Topaz Gigapixel AI**: AI upscaling, resolution enhancement
- **Remini**: Detail enhancement
- **Remove.bg**: Background removal

### Optimization Workflow

**Standard Post-Processing Flow**:
```
1. Basic Adjustments (5 minutes)
   - Fine-tune exposure, contrast, saturation
   - White balance correction

2. Detail Optimization (10 minutes)
   - Sharpen key areas
   - Noise reduction (if necessary)
   - Fix minor flaws

3. Creative Adjustments (5-15 minutes)
   - Add text/logo
   - Color grading (LUT/filters)
   - Composition re-cropping

4. Export Optimization
   - Format selection (JPEG/PNG/WebP)
   - Compression quality balance (file size vs image quality)
```

### Optimization Focus by Use Case

| Use Case | Optimization Focus | Recommended Format |
|----------|-------------------|-------------------|
| **Social Media** | Vibrant colors, strong contrast | JPEG (80-90% compression) |
| **Print Materials** | High resolution, CMYK color | TIFF/PNG |
| **Website Banner** | File size, loading speed | WebP |
| **E-commerce Product** | Clear details, clean background | PNG (transparent background) |
| **Poster** | Large size, high clarity | PDF/PNG |
